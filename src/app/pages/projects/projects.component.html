Animal Help Uruguay

This site was made for an animal shelter, directed by Mary Brown, the site is made with different functions, such as creating an account to create publications of lost pets, which will have to be paid for make the publication. Also on the site you can find information on how donate and can be updated with constant news on the website.

Adopta Ya Uruguay

The site was about adoption and recovery of lost pets, there people they could register their personal account, and start posting their pets on adoption or search for a lost pet. Then the visitors who don't wanted to publish could access the site and see all the pets that were published for adoption, or lost, filtering according to your interest.

Altrion Sites

Altrion Sites is a separate backend/frontend project using an API built in Laravel and then consumed in the frontend built in Angular. The project consists of a dashboard to manage elements of a website such as: news, completed projects, services offered. In addition, JWT is used to manage users and tokens for login and to be able to use the API.
